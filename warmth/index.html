Warmth

by `impulse

  • Description
  • Configs
  • License

Adds a warmth dependency for characters.

Servers can customize usage of this plugin with these settings

Warmth Damage (default: 2)

How much damage to take per tick when at 0 warmth.

Warmth Enabled (default: false)

Whether or not the warmth system is enabled.

Warmth Fire Scale (default: 2)

How much warmth you regain by being near an entity that produces warmth (i.e a fire).

Warmth Kill (default: false)

Whether or not to kill characters if they reach zero warmth.

Warmth Loss Time (default: 5)

How many minutes it takes for a player to lose all their warmth.

Warmth Min Temp (default: 10)

The minimum temperature in celsius before you start to lose warmth.

Warmth Recover Scale (default: 0.25)

How much warmth you regain by being indoors. This is a multiplier of warmthLossTime.

Warmth Temp (default: 20)

The current temperature in celsius.

Warmth Tick Time (default: 5)

How many seconds to wait before calculating each player's warmth. You should usually leave this as the default.
