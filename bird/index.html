Bird

by Arny

  • Description
  • Configs
  • License

Adds a bird faction.

Installation

  • Download the zip
  • Extract the zip into your schemas plugin folder

Features

  • Bird faction
  • Administrator config, easily change how birds work
Servers can customize usage of this plugin with these settings

Bird Actions (default: true)

Allow the birds use /me and /it?

Bird Allow Item Interact (default: true)

Allow a bird to interact with items.

Bird Chat (default: true)

Allow the birds to talk?

Bird Death Sounds (default: true)

Play sounds upon bird injure/death.

Bird Flight Speed (default: 100)

The speed at which a bird can fly.

Bird Health (default: 2)

The default health of birds.

Bird Inventory (default: true)

Shrink the birds inventory? (reconnect/server restart may be required)

Bird OOC (default: true)

Allow the birds use OOC?

Bird Recognise Eachother (default: true)

Can birds recognise eachother?.
