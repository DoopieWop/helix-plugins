Door Extensions

by Zoephix

  • Description
  • Commands
  • License

A extension for the doors plugin, adding easy commands that target every door.

Players can use these commands to interact with this plugin

/DoorSetDisabledAll

Disallows any commands to be ran on every door.

/DoorSetHiddenAll

Hides the description of every door, but still allows it to be ownable.

/DoorSetOwnableAll

Makes every door ownable.

/DoorSetUnownableAll

Makes every door unownable.
