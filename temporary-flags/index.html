Temporary Flags

by wildflowericecoffee

  • Description
  • Commands
  • License

This plugin allows server staff to temporarily give flags to a player that automatically expire over time. To use this, you need the ‘Player Flags’ plugin installed and enabled, which can be found here: https://plugins.gethelix.co/player-flags/.

Players can use these commands to interact with this plugin

/PlyGiveTempFlag

Gives a player temporary flags.
