Detailed Descriptions

by Zoephix

  • Description
  • Commands
  • License

Adds the ability for players to create detailed descriptions, which can be examined.

Players can use these commands to interact with this plugin

/CharDetDesc

Sets your detailed description.

/SelfExamine

Examines your detailed description.
