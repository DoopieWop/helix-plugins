Whitelist

by wildflowericecoffee

  • Description
  • Commands
  • Configs
  • License

This plugin adds a configurable server whitelist, letting only the players you allow to connect to your server. The whitelist is disabled by default, to turn it on simply go to the “config” tab in-game.

Players can use these commands to interact with this plugin

/WhitelistAdd

Adds a SteamID to the whitelist

/WhitelistAddAll

Whitelists all current players

/WhitelistClear

Clears the whitelist

/WhitelistRemove

Removes a SteamID from the whitelist

Servers can customize usage of this plugin with these settings

Whitelist Enabled (default: false)

Enables the server whitelist.
